# solana-nft-verifier

Verify NFT ownership, metadata, and collection membership on Solana using Metaplex and Helius DAS API.

## Problem
NFT verification on Solana requires checking multiple sources. This skill streamlines the process.

## Install
```bash
./install.sh
```

## Usage
```bash
# Verify NFT ownership
node verify.js --mint <address> --owner <wallet>
```
